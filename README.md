# 🚀 Lab Money

Este projeto é uma aplicação web de finanças pessoais, desenvolvida para ser a nossa base de aprendizado e aplicação prática dos conceitos de React. Ao longo das próximas semanas, iremos evoluir esta aplicação, transformando-a em uma ferramenta funcional para registrar e gerenciar entradas e saídas de transações.

O intuito é que o Lab Money sirva como um projeto "vivo" onde cada novo conceito aprendido será imediatamente aplicado, deixando a aplicação cada vez mais robusta e funcional.

---

## 👨‍🏫 Conceitos que Iremos Aplicar

* **Fundamentos do React:** Componentes funcionais, JSX e props.
* **Gerenciamento de Estado:** Uso dos Hooks `useState` e `useEffect` para controlar o estado da aplicação.
* **Comunicação entre Componentes:** Fluxo de dados "pai para filho" (props) e eventos "filho para pai".
* **Componentes Controlados:** Formulários de registro de transações.
* **Estilização e Design:** Aplicação de `CSS Modules` e a filosofia do **Atomic Design** na organização dos componentes.
* **Roteamento:** (`React Router DOM`) para navegação entre diferentes telas (login, home, nova transação).
* **Gestão de Estado Global:** (Tópico futuro) Utilização de `Context API` para compartilhar dados entre todos os componentes.
* **Boas Práticas:** Imutabilidade e criação de Hooks customizados para reutilizar a lógica.

---

## 💡 Funcionalidades que Iremos Construir

* **Login:** Uma tela inicial para simular a autenticação.
* **Dashboard Financeiro:** Visão geral do saldo atual, entradas e saídas.
* **Registro de Transações:** Um formulário para adicionar novas transações.
* **Listagem de Transações:** Uma tabela ou lista detalhada com todas as transações.

## 🚀 Como Rodar o Projeto

Siga estes passos para ter o Lab Money funcionando no seu ambiente de desenvolvimento:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/FuturoDEV-Joinville-V4/m1s10-lab-money.git](https://github.com/FuturoDEV-Joinville-V4/m1s10-lab-money.git)
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd m1s10-lab-money
    ```
3.  **Instale as dependências:**
    ```bash
    npm install
    ```
4.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

A aplicação estará disponível em `http://localhost:5173/` (a porta pode variar).



## 🌳 Fluxo de Trabalho Git

Para este projeto, usaremos um fluxo de trabalho com branches dedicadas a cada aula. Ao final de cada aula, faremos o merge de todas as branches de trabalho para a branch `main`, seguindo os princípios de um `Gitflow` simplificado.
